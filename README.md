# Nixvim Malleus Edition

## Testing your new configuration

To test your configuration simply run the following command
```
nix flake check .
```

## Running Locally
```
nix run .
```

## Running
```
nix run github:speedster33/nixvim --experimental-features 'nix-command flakes'
```
